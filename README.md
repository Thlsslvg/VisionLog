# VisionLog Analytics

## Overview

VisionLog Analytics is an industrial monitoring system developed in Python.

The application reads inspection logs generated by vision systems, stores rejection data in a SQLite database, monitors new logs in real time using Watchdog, and provides visualization and analysis through a Streamlit dashboard.

---

## Features

* Read `.txt` inspection logs
* Parse camera, status, defect and time information
* Store data in SQLite database
* Prevent duplicate log processing
* Real-time monitoring using Watchdog
* Interactive dashboard
* Analytics and statistics
* Log filtering
* CSV and Excel export
* Automatic database creation
* Automatic dashboard refresh

---

## Log Format

Each log file must follow this format:

```txt
Camera : 01
Status : Reject
Defect : Glue
Time   : 16:39
```

## Installation

Install all required libraries:

```bash
pip install databases
pip install aiosqlite
pip install watchdog
pip install streamlit
pip install streamlit-autorefresh
pip install pandas
pip install plotly
pip install pillow
pip install openpyxl
```

---

## Running the Application

### Start Backend

```bash
python main.py
```

The backend will:

* Create the database automatically
* Create the table automatically
* Process existing logs
* Monitor new logs in real time

### Start Dashboard

Open another terminal:

```bash
streamlit run dashboard/App.py
```
